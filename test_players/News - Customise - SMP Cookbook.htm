Customising News Recipe

Each recipe consists of bootstrap settings and playlist options, plus some extra functionality to demonstrate these combinations. This may be in the form of content switchers, some input fields that interact with the player, plugins or more. Below you can see the settings and playlist currently being used. Feel free to edit them and click "Try this out" to see the effect your changes have on the recipe.

Note that these fields accept JSON rather than JS Objects, which means that keys need to be wrapped in quotes as well as values.

Settings

  • aaampGuidance

    The guidance warning displayed before playback when launching BBC Media Player (Android). BBC Media Player does not support guidance through other methods.

    Type: string
  • allowCasting

    Enable UI interactions to send this content to another device, e.g. Chromecast.

    Type: boolean
  • appName

    The name of your app, as you use for DAx tracking, e.g. 'iplayer'. Defaults to 'SMPFlash' or 'SMPHTML5'.

    Type: string
  • appType

    The type of app, as you use for DAx tracking.

    Default: web
    Possible values: web, responsive, mobile-web, mobile-app, bigscreen-html, bigscreen-flash, bigscreen-cast
  • appVersion

    The appVersion reported in the ns_app_ver field. This value is passed to Echo.setAppVersion.

    Type: string
  • attemptHlsPlaybackOnAndroid

    When enabled, attempt to play HLS content on Android when it is available. By default HLS is only attempted as a last resort. Not recommended.

    Please contact the SMP team using the details at the top of this page if you believe you need this option.

    Type: boolean
  • autoplay

    If the content should autoplay or not. Autoplay may or may not be supported, for more information, see [TODO: Link].

    Type: boolean
  • backgroundPlayerFullscreenAllowed

    Allow background player to be embedded in browsers which normally only allow out of page playback. Videos will not play, but can be triggered by the Play API.

    Type: boolean
  • castPlaceholderImage

    When set, if the holding image passed by a playlist file does not contain $recipe or {recipe}, Chromecasts will display this image instead of the one defined in the playlist.

    Type: string
  • continuousPlay

    Enable continuous play.

    Type: boolean
    Default: true
  • counterName

    The page's DAx countername.

    Type: string
  • dashPlaybackOnly

    When set to true, assume that the provided playlist is only available in MPEG-DASH. Playback will not be attempted if the browser does not support DASH, but will always attempt to use a DASH-capable player if it does.

    Please contact the SMP team using the details at the top of this page if you believe you need this option.

    Type: boolean
  • delayEmbed

    Delay embedding the actual player until playback is requested. For SMP Flash only.

    Type: boolean
  • embeddedOffsite

    Set this to true when creating iframe for embedding outside of the BBC (eg. syndication).

    Type: boolean
  • externalEmbedCode

    Boilerplate code provided for embedding. Occurences of '{url}' will be replaced with the externalEmbedUrl.

    Type: string
    Default: <iframe width="400" height="500" frameborder="0" src="{url}"></iframe>
  • externalEmbedUrl

    The url of the page to be used when embedding the content, used in the embed page when embedding playlistObject content, we will substitute {vpid} with the vpid of the playlistObject.

    Type: string
  • forceGuidance

    The guidance warning displayed on all videos, unless the video has an explicit guidance, in which case that is used instead.

    Type: string
  • guidanceCleared

    Treat guidance as a warning rather than click-through. Defaults to false for news product, true for iplayer product.

    Type: boolean
  • under16

    Triggers the guidance flow for an under 16 registered user

    Type: boolean
  • over16

    Triggers the guidance flow for an over 16 registered user

    Type: boolean
  • includeAppClickHandler

    Include the click handler to trigger playback on the container for apps. If this is false the only way to trigger playback is with a custom .play() call.

    Type: boolean
    Default: true
  • insideIframe

    When set to true, it indicates that the player is inside an IFRAME. When the player needs to navigate to another page, it will produce an event "navigateTo", this needs to be bound to by the page and a navigation to the href in the event performed on the top.location, due to browser security.

    Type: boolean
  • loop

    Loop the content. Default is true for the 'background' product. Not supported in BBC Media Player.

    Type: boolean
  • lowBandwidth

    Limit playback to lowest quality streams. See Quality options for more information.

    Type: boolean
  • maintainTheLock

    When embedded on an https page, SMP will not attempt to play several legacy formats in order to maintain security, causing errors for the user.

    Type: boolean
  • mediaProfile

    DEPRECATED

    The collection of mediasets to use for getting content. Currently, pc for iPlayer content or journalism for everything else, but workflow changes should make pc the default. Default depends on product.

    Possible values: pc, journalism
  • mediator

    These settings allow you to change how SMP discovers the media streams you wish to play. In almost all cases, SMP makes the correct decision about what to use - so this is generally only for testing purposes.

    Please contact the SMP team using the details at the top of this page if you believe you need this option.

    Type: object
    • mediator.fck

      Force connection kind. Use anything that will fully or partially match a supplier id from Media Selector.

      Type: string
    • mediator.host

      The mediaselector host to use. By default, it will use the one that matches the BUMP environment. Test mediaselector for test BUMP, stage mediaselector for stage BUMP, and live mediaselector for live BUMP.

      Type: string
      Default: open.${ENV}.bbc.co.uk
    • mediator.href

      Completely override the default mediation URL builder with the provided value. Makes the values of mediator.host, mediator.saml, mediator.overrideMediaSet and VPIDs in the playlistObject have no effect on content played back. Default is constructed by players, e.g. http://open.live.bbc.co.uk/mediaselector/5/select/version/2.0/vpid/bbc_one_london/format/xml/mediaset/pc/atk/${TOKEN}

      Type: string
    • mediator.ms6percentage

      A value between 0 and 1 of the proportion that the player should try using Media Selector 6. So 1 is always, 0 is never, and 0.5 is 50%.

      Type: number
    • mediator.overrideMediaSet

      Override the default decision logic for deciding the mediaset. Use when your content is only available in a specific mediaset (e.g. podcasts).

      Type: string
    • mediator.saml

      The mediaselector host to use when a SAML-authentication is required for content. It is not set by default.

      Type: string
  • muted

    Whether or not to start playback muted.

    Type: boolean
  • noAutomaticNextQueued

    Set this to prevent continuing to the next item on the playlist. User will instead be left paused at the end of the current video and can progress either manually using the next button, or via API using .next()

    Type: boolean
  • noTracking

    SMP will disable stats. You must set this if the user has opted out of reporting according to their privacy settings.

    Type: boolean
  • overrideCtaDuration

    Value will replace the duration shown on the call to action button. Defaults to the duration of the first programme on the playlist

    Type: string
  • overrideError

    Set custom messages for errors. Example: 

    {"6010": {"en": "Flash sucks, well done for not having it installed!"}}

    Type: object
  • overrideErrorLink

    Set the link used on the Flash install error (6010). Example: 

    {"linkTitle": {"en": "I am a bad person and want Flash."}, "link": "https://get.adobe.com/flashplayer/"}

    Type: object
  • overrideHoldingImage

    Override the holding image defined in the playlist. The string '{recipe}', if included, will be automatically chef'd out.

    Type: string
  • pageType

    DEPRECATED

    Used for FMTJ stats, as currently used by journalism.

    Possible values: eav1, eav2, eav3, eav4, eav5, eav6

    DEPRECATED

    Used for FMTJ stats, this is the CPS parent section id.

    Type: number

    Type: boolean
  • playlist

    DEPRECATED

    The URL to an XML playlist.

    Type: string
  • playlistObject

    See the playlistObject reference.

    Type: object
  • plugins

    DEPRECATED

    A legacy method for loading plugins. loadPlugin() should be used.

    Type: object
    • plugins.player

      DEPRECATED

      An object that describes the player.

      Type: object
    • plugins.smpAirPlugins

      DEPRECATED

      The identifier for BBC Media Player (Android) plugins to be used. If you are using plugins for this player, the value of this property will be provided to you by the Media Playout team.

      Type: string
    • plugins.toLoad

      An array of objects describing plugins to be loaded.

      Type: array
  • preferHtmlOnDesktop

    When enabled, prefer the HTML5 player over SMP Flash on desktop.

    Type: boolean
  • preferHtmlOnMobile

    When enabled, prefer the HTML5 player over BBC Media Player on Android.

    Type: boolean
  • preferHtmlControls

    Use the hybrid player (playback can be in flash, but it lives inside the HTML5 player so that the UX looks like the HTML player and html plugins are supported, this is currently only supported if you pre-agree it with the SMP team, and comes with some caveats.

    Type: boolean
  • product

    Each product applies a default group of settings. For example iPlayer sets the foreground colour to iPlayer Pink. You can actually put anything you like here, but if you don't use one of the defaults, you'll have to supply your own custom settings.

    Default: iplayer
    Possible values: iplayer, news, sport, background
  • promptGuidanceWhenAutoplay

    When enabled, autoplaying will show the guidance prompt immediately.

    Type: boolean
  • quality

    Controls the quality shown to the user. See Quality options for more information.

    Default: medium
    Possible values: low, medium, high, hd
  • requestWMP

  • responsive

    When enabled, the player will fill the container it is placed in. Otherwise, width() and height() need to be manually specified.

    Type: boolean
  • significantTime

    The interval, in seconds, in which to send significantTimeUpdates

    Default: 10
  • siteID

    DEPRECATED

    Possible values: iPlayer, News, Sport, Weather
  • startTime

    Start playback at this time (counted as seconds into the programme). Anything greater than 0 skips any idents or competition warnings.

    Type: number
  • statsObject

    Required for optimal stats reporting.

    Type: object
    • statsObject.clipPID

    A list of one or more pieces of content to play. Should only contain one main programme for stats purposes. If you wish to play multiple 'important' assets, queue a playlist for each.

      Specify a Content Version ID (CVID) to play (a CVID is a version ID that doesn't depend upon PIPs).

      Type: string

      The length of the item, in seconds.

      Type: number

      An array of connections of varying transfer formats and/or suppliers.
